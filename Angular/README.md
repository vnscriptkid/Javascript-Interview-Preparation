## Angular CLI
```console
npm install -g @angular/cli
```

## Create new app
```console
ng new my-app
```

## Import 3rd-party CSS
```console
yarn add bulma
yarn add bootstrap
yarn add semantic-ui-css
```

```css
// src/styles.css
@import "bulma/css/bulma.css";
@import "bootstrap/dist/css/bootstrap.css";
@import "semantic-ui-css/semantic.css";
```
